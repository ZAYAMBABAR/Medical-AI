# Medical AI - Disease Detection & AI Assistant

A Medical AI application developed as part of a course project. The system uses Deep Learning models to analyze medical images and provides AI-powered assistance using the Groq API.

> **Disclaimer:** This project is for educational purposes only and should not be used for real medical diagnosis.

---

##  Features

- Bone Fracture Detection
- Brain MRI Analysis
- Chest X-ray Classification
- Dental Image Analysis
- AI Medical Chat Assistant (Groq API)
- Image Upload Support
- FastAPI Backend
- Simple HTML Frontend

## 🚀 Installation






## Configure the Groq API

Rename or create the `.env` file inside the `backend` folder and add your Groq API credentials.

Example:

```env
GROQ_API_KEY=YOUR_API_KEY
GROQ_BASE_URL=https://api.groq.com/openai/v1
GROQ_MODEL=YOUR_MODEL_NAME
GROQ_CHAT_MODEL=YOUR_CHAT_MODEL
```



---

##  Running the Project

Run the FastAPI application:

```bash
uvicorn backend.app:app --reload
```

Open your browser and visit:

```
http://127.0.0.1:8000
```
